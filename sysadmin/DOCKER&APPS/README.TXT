EN ESTE README VOY A DESCRIBIR CUANTAS DOCKERAPPS QUE SON SOFTWARE LIBRE, SIRVEN COMO HERRAMIENTAS PARA EL SYSADMIN
YA SEA COMO EN SERVIDORES LOCALES O LO EQUIVALENTE A LA NUBE (AWS,GOOGLE CLOUD,ETC)
NO VA A TRATAR DE PROGRAMAR DOCKER APPS O PROGRAMAR EN DOCKER O DOCKER COMPOSE

LAS DOCKERS APPS VIENEN EN DOCKER Y DOCKER COMPOSE
-(ACONSEJO INSTALAR LAS DOCKERAPPS EN MAQUINAS VIRTUALES NO DIRECTAMENTE EN PROXMOX, QUIZAS EN OTRO CONTEXTO SIRVA PARA 
INSTALARLO EN CERCA DEL HOST)

-(OTRO CONSEJO ES QUE DESTINEMOS EL VOLUMEN DE LA DOCKER APP FUERA DEL DEL DISCO PRIMARIO YA QUE SI PASA ALGO CON EL SERVIDOR 
  TODA LA DATA SE PERDERIA, INSTALEMOS LAS DOCKERAPPS EN UN DISCO SECUNDARIO, Y HACER BACKUP PERIODICAMENTE)
