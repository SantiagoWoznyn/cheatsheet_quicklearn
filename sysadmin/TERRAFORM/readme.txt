Terraform no es un lenguaje de programación en el sentido tradicional, 
sino una herramienta de infraestructura como código (IaC, por sus siglas en inglés). Utiliza su propio lenguaje de configuración 
llamado HCL (HashiCorp Configuration Language), que se utiliza para definir recursos de infraestructura como servidores, redes, bases de datos, 
etc., en un formato declarativo.

En resumen, HCL es un lenguaje de configuración específico para definir la infraestructura, 
mientras que Terraform es la herramienta que interpreta esos archivos para gestionar y automatizar la provisión de infraestructura.
